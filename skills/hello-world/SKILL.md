# Hello World Skill

A simple example skill for the Kimi Browser CLI.

## What it does

This skill demonstrates how skills work in the marketplace system.

## Usage

When you run `kimi-browser skill use hello-world`, this SKILL.md is displayed.

## Capabilities

Skills can include:
- **Instructions** (like this file) - displayed when used
- **Python scripts** (main.py) - executed automatically
- **Configuration files** - JSON, YAML, etc.
- **Resources** - templates, data files, etc.

## Creating Your Own Skill

1. Create a directory in `~/.kimi/skills/<your-skill>/`
2. Add a `SKILL.md` with instructions
3. Optionally add a `main.py` for executable skills
4. Create a `manifest.json` with metadata

## Publishing

To publish to SkillsMP marketplace:
1. Create an account at skillsmp.com
2. Package your skill files
3. Upload via the API or web interface
